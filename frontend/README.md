# Frontend Zwitter

### Create Project
```bash
npx create-next-app
```

### Install Dependencies
```bash
npm i @fortawesome/react-fontawesome @fortawesome/fontawesome-svg-core @fortawesome/free-solid-svg-icons @fortawesome/free-regular-svg-icons @fortawesome/free-brands-svg-icons
```

### Running Project

```bash
npm run dev
```
